#  Proyecto Plataforma de Exámenes CUTE

Bienvenido/a al Proyecto! Este es un mini proyecto **open source** diseñado para mejora la forma en que practicamos para nuestros parciales. 

## La Misión
El objetivo principal de esta página es transformar la aburrida y estresante tarea de estudiar en una experiencia interactiva, divertida y con mucho estilo. Con una estética mas clean, buscamos que el repaso de temas y la resolución de exámenes de práctica se sientan menos como una obligación y más como un juego.

## ¿Quién es Vivi?
**Vivi** no es solo una mascota, es tu compañera virtual de estudio. 
Vivi está integrada en la plataforma para acompañarte mientras resuelves tus exámenes. Ella te dará feedback interactivo y personalizado, celebrará tus aciertos y te animará cuando te equivoques, todo con un tono relajado, divertido y muy Gen Z. Su propósito es hacerte sentir acompañado/a y motivado/a durante todo el proceso de aprendizaje.

## Cómo usar la plataforma

*(Nota: Si descargaste la versión compilada en `.exe` (Esta en el repositorio), ¡es aún más fácil! Solo haz doble clic en el ejecutable y la aplicación se abrirá sola, sin necesidad de usar la terminal).*

El proyecto cuenta con una interfaz web y un pequeño servidor en Python para manejar ciertas funcionalidades (como el procesamiento de exámenes). Para ejecutarlo localmente, sigue estos pasos:

1. **Requisitos previos:** Asegúrate de tener instalado [Python](https://www.python.org/downloads/) en tu computadora.
2. **Instalar dependencias:** 
   Abre tu terminal en la carpeta del proyecto e instala las librerías necesarias. Generalmente se necesita Flask:
   ```bash
   pip install flask
   ```
3. **Iniciar el servidor:**
   Ejecuta el archivo del servidor para levantar el backend:
   ```bash
   python server.py
   ```
4. **Abrir la aplicación:**
   Una vez que el servidor esté corriendo, abre tu navegador web y ve a la dirección local que te indique la terminal (por ejemplo, `http://localhost:5000` o `http://127.0.0.1:5000`).
5. **¡A estudiar!** Ingresa tu nombre, elige tu modo y deja que Vivi te guíe en tu sesión de estudio.



## 🤝 ¡Ayudas y Mejoras son Bienvenidas!
